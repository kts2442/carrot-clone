## Prisma

Node.js와 Typescript ORM(Object Relational Mapping)

- Prisma를 사용하기 위해서는 먼저 Prisma에게 DB가 어떻게 생겼는지, 데이터의 모양을 설명해줘야 함 => schema.prisma
- Prisma를 통해서 clinet가 js객체의 instance를 생성하고, 이를 통해 db와 직접 상호작용, 자동완성 제공
