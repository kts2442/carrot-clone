## API route

- API route는 Next.js로 API를 빌드하기 위한 솔루션을 제공합니다.
- pages/api 폴더 내의 모든 파일은 /api/\*에 매핑되며 API endpoint로 처리됩니다.
- server-side 전용 번들이며 client-side 번들 크기를 늘리지 않습니다.
