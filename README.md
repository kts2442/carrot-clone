## 1.detail HTML

HTML details emlments는 열고 닫힘을 통해 내용을 visible 유무 기능 제공

```
<details>
  <summary className="cursor-pointer">
    What is my fav.food
  </summary>
  <span className="selection"bg-indigo-300">김치</span>
</details>
```
